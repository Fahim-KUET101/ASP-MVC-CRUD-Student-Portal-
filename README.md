# Student Management Portal (ASP.NET MVC)

A professional web application designed for structured data management, built using the **ASP.NET MVC** architectural pattern. This project serves as a foundational demonstration of building secure, scalable, and responsive CRUD (Create, Read, Update, Delete) systems.

---

## 🚀 Key Features

* **Data Entry (Create):** Efficient onboarding of student records with server-side processing.
* **Record Management (Read):** A centralized dashboard to view all registered students in real-time.
* **Precision Editing (Update):** Update student information dynamically using unique ID-based targeting.
* **Data Lifecycle (Delete):** Secure removal of specific records to maintain database hygiene.
* **Responsive UI:** Styled with **Bootstrap 5** to ensure a seamless experience across mobile, tablet, and desktop browsers.

---

## 🛠️ Technical Stack

* **Backend:** C# / ASP.NET MVC
* **Frontend:** Razor Pages (HTML5, CSS3), Bootstrap 5
* **Architecture:** MVC (Model-View-Controller) Pattern
* **Version Control:** Git / GitHub

---

## 📁 Project Structure

| Folder | Description |
| :--- | :--- |
| **Models** | Contains the data structures and student entity logic. |
| **Controllers** | Handles the business logic and coordinates data flow between Model and View. |
| **Views** | Razor-powered templates for the User Interface. |
